Authentication

+

Endpoint-endpoint yang berhubungan dengan autentikasi user.

Login

POST /auth/login

Authenticate a user and return a JWT token.

Request Body:
{
    "email": "string",
    "password": "string"
}

Response:
{
    "message": "Login successful",
    "token": "JWT_TOKEN"
}

User Management

+

Endpoint yang dapat digunakan untuk manajemen data user

Get Users

GET /user

Retrieve all user data. Requires an admin role and valid JWT token.

Headers:
Authorization: Bearer {token}

Response:
[
    {
        "userid": "UUID",
        "fullname": "string",
        "email": "string",
        "pass": "string"
        "userphone": "string",
        "roleid": "string"
    }
]
